# `penguins setup`

Initialize `~/.penguins/penguins.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
penguins setup
penguins setup --workspace ~/.penguins/workspace
```

To run the wizard via setup:

```bash
penguins setup --wizard
```
