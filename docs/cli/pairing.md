# `penguins pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
penguins pairing list whatsapp
penguins pairing approve whatsapp <code> --notify
```
